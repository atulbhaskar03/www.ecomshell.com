# Ecomshell Soothing Brand v6

Light, soothing redesign based on the approved visual direction.

Deployment:
1. Extract the ZIP.
2. Upload the files inside this folder to the root of the existing GitHub repository.
3. Commit changes.
4. Netlify will redeploy automatically.

Important:
- The founder portrait area is intentionally a branded placeholder.
- Replace it with Atul Soni's original high-resolution professional photograph when available.
- No unverified client logos, testimonials, client counts, or project counts have been included.
